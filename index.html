Wordle-Solver

A few months ago a word guessing game called Wordle had gotten extremely popular. While I was playing the game I became curious, was there a combination of 5 consecutive words that didn't have a single letter in common? Initally I had tried to solve it by selecting words by hand, but it was just too difficult. That is when I started to code an algorithm to solve this problem, eventually coming up with multiple solutions. After this I became interested in creating an website to solve wordles for you, this can be found here.
